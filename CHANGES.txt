0.1 (2012-04-21)
----------------

-  Initial version.
   Ещё довольно сырой продукт.
   Сервер запускается, показывает таблицы из БД,
   доступна сортировка, поиск, постраничный просмотр, просмотр нескольких,
   связанных между собой, таблиц.
   Пользователи и настройки баз данных хранятся в текстовом виде
   (ndt/security.py, ndt/security_db.py).
   После авторизации необходимо ещё инициировать подключение к БД:
   http://localhost:6543/user
   после этого можно работать с просмотром:
   http://localhost:6543/static/view_tables_v1.html
   http://localhost:6543/static/view_tables_v2.html
   http://localhost:6543/static/view_column_district_v1.html
   http://localhost:6543/static/view_column_district_v2.html

   Используются две версии протокола обмена данными - 1ая и 2ая.
   Первая была тестовая и использовала jquery.tablesorter,
   вторая уже заточена под DataTables (который поддерживает обработку данных
   на стороне сервера).

   Тестирование запросов осуществляется через
   http://localhost:6543/static/test.html
   здесь используется вторая версия протокола обмена данными.

   Вики прикручен пока только для ознакомления - есть ли возможность в
   будущем быстро набивать в нём шаблоны для просмотра заданных таблиц
   с заданными параметрами.
